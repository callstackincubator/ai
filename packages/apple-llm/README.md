# Apple LLM

Use Apple's local LLM functionality (Apple Intelligence) in React Native. Standalone, or with Vercel AI SDK.

## Features

### Supported
- Text generation
- Streaming
- Structured output (JSON Schema via Zod)

### Coming Soon
- Tool calling
